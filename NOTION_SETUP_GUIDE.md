# 📘 노션 워크스페이스 설정 가이드

> 닥터푸드 프로젝트를 노션에서 효율적으로 관리하는 방법

---

## 🎯 목표

칸반 보드, 테이블 뷰, 타임라인을 활용한 프로젝트 관리 시스템 구축

---

## 📦 제공 파일

1. **NOTION_TASKS.csv** - 작업 데이터베이스 (30개 작업)
2. **NOTION_DASHBOARD.md** - 프로젝트 대시보드
3. **NOTION_SETUP_GUIDE.md** - 이 가이드

---

## 🚀 노션 설정 단계

### **Step 1: 새 페이지 생성**

1. 노션에서 **새 페이지** 생성
2. 페이지 이름: `닥터푸드 프로젝트`
3. 아이콘 추가: 🏥 또는 🍽️

---

### **Step 2: CSV 파일 Import**

#### 방법 1: 드래그 & 드롭
1. `NOTION_TASKS.csv` 파일을 노션 페이지로 드래그
2. "Create database" 선택
3. 자동으로 데이터베이스 생성됨

#### 방법 2: Import 메뉴
1. 노션 페이지에서 `/database` 입력
2. "Table - Inline" 선택
3. 우측 상단 `⋯` 메뉴 → "Merge with CSV"
4. `NOTION_TASKS.csv` 파일 선택

---

### **Step 3: 데이터베이스 속성 설정**

Import 후 다음 속성이 자동 생성됩니다:

| 속성명 | 타입 | 설명 |
|--------|------|------|
| 작업명 | Title | 작업 이름 |
| 상태 | Select | To Do / 진행중 / 완료 / 보류 |
| 우선순위 | Select | 🔴 High / 🟡 Medium / 🟢 Low |
| 카테고리 | Select | Backend / Frontend / Design / DevOps |
| 시작일 | Date | 작업 시작 날짜 |
| 마감일 | Date | 작업 마감 날짜 |
| 태그 | Multi-select | Auth, API, UI 등 |
| 진행률 | Number | 0-100% |
| 설명 | Text | 작업 상세 설명 |

---

### **Step 4: 칸반 보드 뷰 생성**

1. 데이터베이스 상단의 **"+ Add a view"** 클릭
2. **"Board"** 선택
3. 뷰 이름: `칸반 보드`
4. **Group by**: `상태` 선택
5. **Sort**: `우선순위` (Descending)

#### 결과
```
┌──────────┬──────────┬──────────┬──────────┐
│ 📋 To Do │ 🚧 진행중 │ ✅ 완료  │ 🔒 보류  │
├──────────┼──────────┼──────────┼──────────┤
│ 카드들   │ 카드들   │ 카드들   │ 카드들   │
└──────────┴──────────┴──────────┴──────────┘
```

---

### **Step 5: 테이블 뷰 설정**

1. **"+ Add a view"** → **"Table"**
2. 뷰 이름: `작업 목록`
3. **Sort**: 
   - 1순위: `우선순위` (Descending)
   - 2순위: `마감일` (Ascending)
4. **Filter**: 
   - `상태` is not `완료`

#### 표시할 컬럼
- ✅ 작업명
- ✅ 상태
- ✅ 우선순위
- ✅ 카테고리
- ✅ 마감일
- ✅ 진행률
- ❌ 시작일 (숨김)
- ❌ 설명 (숨김)

---

### **Step 6: 타임라인 뷰 생성**

1. **"+ Add a view"** → **"Timeline"**
2. 뷰 이름: `일정 관리`
3. **Start date**: `시작일`
4. **End date**: `마감일`
5. **Color by**: `우선순위`

#### 타임라인 설정
- **Show weekends**: OFF
- **Zoom**: Week
- **Group by**: `카테고리`

---

### **Step 7: 갤러리 뷰 생성 (선택)**

1. **"+ Add a view"** → **"Gallery"**
2. 뷰 이름: `기능별 보기`
3. **Card preview**: `설명`
4. **Card size**: Medium
5. **Group by**: `카테고리`

---

### **Step 8: 대시보드 추가**

1. 데이터베이스 위에 `/heading` 입력
2. `NOTION_DASHBOARD.md` 내용 복사
3. 노션 페이지에 붙여넣기
4. 자동으로 마크다운 형식 변환됨

---

## 🎨 노션 페이지 구조

최종 페이지 구조:

```
📁 닥터푸드 프로젝트
│
├── 🏠 프로젝트 대시보드 (NOTION_DASHBOARD.md)
│   ├── 📊 프로젝트 현황
│   ├── 🎯 이번 주 목표
│   ├── 🔗 빠른 링크
│   └── 🏗️ 기술 스택
│
└── 📊 작업 데이터베이스 (NOTION_TASKS.csv)
    ├── 📋 칸반 보드 (Board View)
    ├── 📝 작업 목록 (Table View)
    ├── 📅 타임라인 (Timeline View)
    └── 🖼️ 갤러리 (Gallery View)
```

## 💡 활용 팁

### **1. 필터 활용**

#### 이번 주 작업만 보기
```

```

#### 긴급 작업만 보기
```
Filter: 우선순위 is 🔴 High
AND 상태 is not 완료
```

#### 내 작업만 보기
```
Filter: 담당자 is Me
```

---

### **2. 정렬 활용**

#### 우선순위 + 마감일 순
```
Sort: 
1. 우선순위 (Descending)
2. 마감일 (Ascending)
```

#### 카테고리별 그룹화
```
Group by: 카테고리
Sort: 마감일 (Ascending)
```

---

### **3. 템플릿 버튼 생성**

#### 새 작업 템플릿
1. 데이터베이스 상단 `⋯` → "New template"
2. 템플릿 이름: `새 작업`
3. 기본 구조:

```markdown
## 📌 개요
[작업 설명]

## ✅ 체크리스트
- [ ] 할 일 1
- [ ] 할 일 2
- [ ] 할 일 3

## 📚 참고 자료
- [링크 1](#)
- [링크 2](#)

## 💬 메모
[메모 작성]
```

---

### **4. 관계형 데이터베이스 (고급)**

#### 회의록 데이터베이스 연결

2. Relation 속성 추가: `관련 작업`
3. 작업 데이터베이스와 연결

## 🆘 문제 해결

### CSV Import 실패
- **원인**: 파일 인코딩 문제
- **해결**: Excel에서 열어서 "CSV UTF-8"로 저장

### 한글 깨짐
- **원인**: 인코딩 문제
- **해결**: 메모장에서 "UTF-8"로 저장

### 속성이 안 보임
- **원인**: 뷰 설정에서 숨김 처리됨
- **해결**: Properties → Show all
