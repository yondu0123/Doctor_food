## 😰 불편함과 고민 (Pain Points)

### 1. 운영 관리의 복잡성
```
"매일 해야 할 일이 너무 많아요"
```
- 식사 기록, 투약 체크, 건강 상태 확인
- 보호자 연락, 직원 관리, 회계 처리
- 정부 보고서 작성, 평가 준비
- **문제**: 하루 12시간 이상 근무, 주말도 없음

### 2. 인력 부족과 높은 인건비
```
"직원을 더 뽑고 싶지만 여유가 없어요"
```
- 요양보호사 1명당 4명 담당 (법정 기준)
- 야간 근무자 구하기 어려움
- 인건비가 전체 비용의 70%
- **문제**: 인력 충원 vs 수익성 딜레마

### 3. 수기 기록의 한계
```
"종이 서류가 산더미예요"
```
- 식사 일지, 투약 기록, 건강 체크리스트
- 보호자 상담 일지, 사고 보고서
- 정부 제출 서류 (월간, 분기별)
- **문제**: 기록 누락, 분실, 찾기 어려움

### 4. 보호자 민원 대응
```
"보호자님들 전화가 하루에 10통 이상이에요"
```
- "오늘 식사는 잘 하셨나요?"
- "약은 제때 드셨나요?"
- "어제 넘어지셨다는데 괜찮나요?"
- **문제**: 반복적인 질문, 실시간 답변 어려움

### 5. 정부 평가 부담
```
"평가 준비하느라 한 달이 날아가요"
```
- 연 1회 정부 평가 (등급 결정)
- 서류 미비 시 감점
- 등급 하락 시 입소자 감소
- **문제**: 평가 준비에 과도한 시간 소요
