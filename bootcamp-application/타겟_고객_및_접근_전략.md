# 닥터푸드 타겟 고객 및 시장 접근 전략

## 📊 고객 세그먼트 분석

### 🎯 Primary Customer (주 고객): 요양원 운영자

#### 왜 요양원 운영자가 주 고객인가?

**1. 의사결정권자 (Decision Maker)**
```
✅ 구매 결정권: 요양원 운영자가 SaaS 구독 결정
✅ 예산 권한: 월 구독료 지불 주체
✅ 도입 주도: 시스템 도입 및 직원 교육 주관
```

**2. 가장 큰 Pain Point 보유**
```
💰 비용 압박
- 인건비 증가 (간병인 1인당 월 250만원)
- 식자재 비용 상승
- 정부 지원금 감소

⏰ 운영 효율 저하
- 하루 2시간 이상 기록 업무
- 보호자 응대로 인한 업무 중단
- 투약 실수로 인한 법적 리스크

📉 경쟁 심화
- 신규 요양원 증가
- 보호자 선택권 확대
- 평판 관리 중요성 증가
```

**3. 명확한 ROI (투자 대비 효과)**
```
비용 절감:
- 기록 시간 75% 단축 → 인건비 절감
- 식자재 공동구매 38% 절감
- 투약 실수 방지 → 법적 비용 절감

매출 증가:
- 보호자 만족도 향상 → 입소자 증가
- 정부 평가 A등급 → 지원금 증가
- 디지털 전환 → 브랜드 이미지 개선
```

## 🎯 고객 세그먼트별 우선순위

### 1순위: 중소형 요양원 (입소자 30-50명)

**특징**:
- 전국 3,500개 (전체의 60%)
- 운영자가 직접 현장 관리
- 디지털 전환 필요성 높음
- 예산 제약 있지만 효율화 절실

**접근 전략**:
```
✅ 무료 체험 14일 제공
✅ 월 구독료 ₩9,900/인 (부담 없는 가격)
✅ 설치 없이 웹/앱으로 즉시 사용
✅ 1:1 온보딩 지원
```

**예상 전환율**: 15-20%

---

### 2순위: 프랜차이즈 요양원 체인

**특징**:
- 5-10개 지점 운영
- 표준화된 운영 프로세스 필요
- 높은 예산 및 의사결정 속도

**접근 전략**:
```
✅ 엔터프라이즈 플랜 제공
✅ 다수 지점 통합 관리 기능
✅ 본사-지점 간 데이터 공유
✅ 맞춤형 계약 및 할인
```

**예상 전환율**: 30-40% (한 번 도입 시 전 지점 확산)

---

### 3순위: 신규 개원 요양원

**특징**:
- 연간 200-300개 신규 개원
- 처음부터 디지털 시스템 도입 가능
- 레거시 시스템 없음

**접근 전략**:
```
✅ 개원 패키지 제공
✅ 초기 3개월 50% 할인
✅ 개원 컨설팅 지원
✅ 정부 지원 사업 연계
```

**예상 전환율**: 40-50% (레거시 없어 도입 용이)

## 🚀 시장 접근 전략 (Go-to-Market Strategy)

### Phase 1: 초기 진입 (0-6개월)

#### 1. 파일럿 고객 확보 (3개 요양원)

**타겟**:
- 서울/경기 중소형 요양원
- 디지털 전환에 관심 있는 운영자
- 입소자 30-40명 규모

**접근 방법**:
```
1. 직접 방문 영업
   - 요양원 협회 네트워킹
   - 지인 소개
   - 콜드 콜 (Cold Call)

2. 무료 체험 제안
   - 14일 무료 체험
   - 설치 없이 즉시 사용
   - 1:1 온보딩 지원

3. 성공 사례 만들기
   - 사용 전/후 비교 데이터
   - 운영자 인터뷰 영상
   - 보호자 만족도 조사
```

**예상 비용**: ₩5M (영업 활동비)
**예상 기간**: 3개월

---

#### 2. 온라인 마케팅

**채널**:
```
1. 네이버 블로그/카페
   - "요양원 운영 노하우" 콘텐츠
   - "디지털 전환 성공 사례"
   - SEO 최적화

2. 유튜브
   - 제품 데모 영상
   - 고객 인터뷰
   - 요양원 운영 팁

3. 페이스북/인스타그램
   - 요양원 운영자 그룹 타겟팅
   - 성공 사례 공유
   - 무료 체험 광고
```

**예상 비용**: ₩3M/월
**예상 전환율**: 2-3%

---

#### 3. 오프라인 이벤트

**활동**:
```
1. 요양원 협회 세미나
   - 부스 운영
   - 제품 데모
   - 명함 교환

2. 요양원 박람회
   - 전시 부스
   - 현장 상담
   - 무료 체험 신청

3. 지역별 간담회
   - 소규모 운영자 모임
   - 제품 소개 및 Q&A
   - 네트워킹
```

**예상 비용**: ₩5M/분기
**예상 전환율**: 10-15%

---

### Phase 2: 성장 가속 (6-18개월)

#### 1. 입소문 마케팅 (Referral)

**전략**:
```
✅ 추천 프로그램
- 기존 고객이 신규 고객 추천 시
- 양쪽 모두 1개월 무료 제공
- 추천 3개 이상 시 영구 할인

✅ 성공 사례 홍보
- 고객 인터뷰 영상 제작
- 언론 보도 자료 배포
- 업계 매체 기고
```

**예상 효과**: 신규 고객의 30-40%가 추천으로 유입

---

#### 2. 파트너십 확대

**파트너**:
```
1. 식자재 공급업체
   - 공동구매 플랫폼 연계
   - 상호 고객 소개
   - 공동 마케팅

2. 요양원 프랜차이즈 본사
   - 전 지점 일괄 도입
   - 맞춤형 기능 개발
   - 장기 계약

3. 정부/지자체
   - 디지털 전환 지원 사업
   - 보조금 연계
   - 공공 입찰 참여
```

**예상 효과**: 월 10-15개 요양원 신규 계약

---

#### 3. 콘텐츠 마케팅

**콘텐츠**:
```
1. 백서 (White Paper)
   - "요양원 디지털 전환 가이드"
   - "AI 식단 관리 효과 분석"
   - "보호자 소통 베스트 프랙티스"

2. 웨비나 (Webinar)
   - 월 1회 온라인 세미나
   - 제품 데모 및 Q&A
   - 업계 전문가 초청

3. 뉴스레터
   - 주간 요양원 운영 팁
   - 신규 기능 소개
   - 고객 성공 사례
```

**예상 효과**: 브랜드 인지도 향상, 리드 생성

---

### Phase 3: 시장 지배 (18-36개월)

#### 1. 대형 체인 공략

**타겟**:
- 10개 이상 지점 운영 체인
- 연간 매출 100억원 이상
- 전국 네트워크 보유

**접근 방법**:
```
✅ 엔터프라이즈 영업팀 구성
✅ 맞춤형 제안서 작성
✅ POC (Proof of Concept) 진행
✅ 장기 계약 (3-5년)
```

**예상 효과**: 1개 체인 = 10-20개 요양원

---

#### 2. 지역별 확산

**전략**:
```
1. 서울/경기 (1년차)
   - 초기 집중 공략
   - 성공 사례 축적

2. 부산/대구/광주 (2년차)
   - 지역별 영업 담당자 배치
   - 지역 파트너 발굴

3. 전국 확산 (3년차)
   - 온라인 마케팅 강화
   - 자동화된 온보딩
```

**예상 효과**: 전국 100개 요양원 확보

## 💰 고객 획득 비용 (CAC) 및 생애 가치 (LTV)

### CAC (Customer Acquisition Cost)

```
채널별 CAC:
- 직접 영업: ₩1.5M/고객
- 온라인 광고: ₩800K/고객
- 추천: ₩300K/고객 (가장 효율적)
- 파트너십: ₩500K/고객

평균 CAC: ₩700K/고객
```

### LTV (Lifetime Value)

```
계산:
- 월 구독료: ₩400K (40명 × ₩10K)
- 평균 사용 기간: 36개월
- 이탈률: 10%/년

LTV = ₩400K × 36개월 × 0.9 = ₩12.96M

LTV/CAC 비율 = 18.5 (매우 건강한 수치)
```

## 🎯 핵심 메시지 (Value Proposition)

### 요양원 운영자에게

**"하루 2시간 절약하고, 월 100만원 아끼세요"**

```
✅ 기록 시간 75% 단축
   → 간병인이 실제 돌봄에 집중

✅ 식자재 비용 38% 절감
   → 공동구매로 월 100만원 절약

✅ 보호자 만족도 80% 향상
   → 입소 대기자 증가

✅ 정부 평가 A등급 달성
   → 지원금 증가
```

---

### 간병인에게

**"더 이상 기록에 시간 낭비하지 마세요"**

```
✅ 사진 한 장으로 식사 기록 완료
✅ 투약 알림으로 실수 방지
✅ 보호자 문의 자동 응답
✅ 스마트폰으로 언제 어디서나
```

---

### 보호자에게

**"부모님 상태를 실시간으로 확인하세요"**

```
✅ 오늘의 식사 사진
✅ 투약 완료 알림
✅ 건강 상태 리포트
✅ 간병인과 1:1 채팅
```

## 🎯 결론

### 가장 큰 소비자층: 요양원 운영자 (중소형 30-50명 규모)

**이유**:
1. ✅ 의사결정권자 (구매 결정)
2. ✅ 가장 큰 Pain Point (비용, 효율, 경쟁)
3. ✅ 명확한 ROI (비용 절감 + 매출 증가)
4. ✅ 시장 규모 (전국 3,500개)

### 접근 전략:

**1단계 (0-6개월)**: 파일럿 고객 확보
- 직접 영업 + 무료 체험
- 성공 사례 만들기

**2단계 (6-18개월)**: 입소문 확산
- 추천 프로그램
- 파트너십 확대
- 콘텐츠 마케팅

**3단계 (18-36개월)**: 시장 지배
- 대형 체인 공략
- 전국 확산
- 브랜드 리더십

### 핵심 성공 요인:

```
✅ 무료 체험으로 진입 장벽 낮추기
✅ 성공 사례로 신뢰 구축
✅ 추천 프로그램으로 입소문 확산
✅ 파트너십으로 빠른 성장
✅ 명확한 ROI로 재계약 유도
```
