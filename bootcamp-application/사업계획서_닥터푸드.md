## 1. 요약 (Executive Summary)

### 🎯 한 줄 소개
**닥터푸드는 요양원 운영자, 보호자, 간병인을 하나의 플랫폼으로 연결하여 식사·투약·건강 관리를 자동화하는 AI 기반 통합 케어 솔루션입니다.**

### 💡 핵심 가치 제안
- **사진 한 장으로 식사 기록 완료** → 기록 시간 75% 단축
- **AI 개인 맞춤 식단 추천** → 32명 전원의 식단을 자동 생성
- **보호자 앱 실시간 공유** → 전화 문의 70% 감소

### 📊 시장 기회
- **국내 시장**: 요양원 5,800개, 입소자 23만명
- **글로벌 시장**: 미국 시니어케어 시장 $436B (2023)
- **성장률**: 연평균 7.2% 성장

### 💰 비즈니스 모델
- **B2B SaaS**: 입소자 1인당 월 9,900원 구독
- **B2B2C 확장**: 보호자 프리미엄 기능 유료화
- **부가 매출**: AI 리포트, 식자재 공동구매 제휴

## 2. 문제 정의 (Problem)

### 🏥 요양원 운영의 3가지 고통

#### 문제 1: 기록 업무 과중
```
현황:
- 하루 2시간 이상 수기 작성
- 식사 일지, 투약 기록, 건강 체크리스트
- 간병인 업무의 40%가 기록 작성

결과:
→ 실제 돌봄 시간 부족
→ 인건비 증가
→ 간병인 피로도 증가
```

#### 문제 2: 보호자 응대 부담
```
현황:
- 하루 10통 이상의 전화 문의
- "어머니 식사 잘 하셨나요?" 반복 질문
- 업무 중단으로 인한 효율 저하

결과:
→ 간병인 스트레스 증가
→ 보호자 불안감 지속
→ 신뢰 구축 어려움
```

#### 문제 3: 투약 누락 위험
```
현황:
- 12명의 어르신, 각각 3가지 약
- 체크리스트 없이는 실수하기 쉬움
- 투약 누락 시 건강 악화

결과:
→ 법적 책임 문제
→ 가족 불만 증가
→ 요양원 평판 하락
```

### 📉 시장 조사 결과

**요양원 운영자 인터뷰 (N=15)**
- 기록 업무 부담: 93%
- 보호자 응대 스트레스: 87%
- 투약 관리 어려움: 73%
- 디지털 솔루션 필요성: 100%

## 3. 솔루션 (Solution)

### 🚀 닥터푸드의 3가지 핵심 기능

#### 기능 1: 사진 한 장으로 식사 기록 자동화
```
Before (수기 작성):
1. 식사 사진 촬영
2. 메뉴 수기 작성
3. 섭취량 기록
4. 특이사항 메모
→ 1인당 5분 소요 × 32명 = 160분/일

After (닥터푸드):
1. 사진 촬영
2. AI 자동 분석 완료
→ 1인당 30초 × 32명 = 16분/일

절감 효과: 144분/일 (90% 단축)
```

#### 기능 2: AI 개인 맞춤 식단 추천
```
입력:
- 환자 건강 데이터 (당뇨, 신장투석, 고혈압 등)
- 기저질환별 영양 요구량
- 공통 식자재 풀

출력:
- 32명 전원의 개인 맞춤 식단
- 조리법만 다르게 (비용 최적화)
- 영양 균형 보장

효과:
→ 영양사 인력 비용 절감
→ 식자재 공동구매로 38% 비용 절감
→ 정부 평가 A등급 달성
```

#### 기능 3: 보호자 앱 실시간 공유
```
보호자가 확인 가능한 정보:
✅ 오늘의 식사 (사진 + 메뉴)
✅ 투약 현황 (완료/미완료)
✅ 건강 상태 (혈압, 혈당 등)
✅ 활동 사진 (산책, 여가 활동)
✅ 간병인과 1:1 채팅

효과:
→ 전화 문의 70% 감소 (10통 → 3통)
→ 보호자 불안감 80% 감소
→ 요양원 신뢰도 향상
```

```

- 연 매출: ₩720M
```

```
보호자 프리미엄 기능:
- 월간 상세 리포트: ₩5,000/월
```

```
✅ 3-Role 통합 플랫폼 (유일)
```
