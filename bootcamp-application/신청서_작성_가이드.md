## 📄 PDF 변환 방법

### 방법 1: VS Code 확장 프로그램
```
1. VS Code에서 Markdown Preview Enhanced 설치
2. Markdown 파일 열기
3. 우클릭 → "Markdown Preview Enhanced: Export (PDF)"
```

### 방법 2: 온라인 변환 도구
```
1. https://www.markdowntopdf.com/ 접속
2. Markdown 파일 업로드
3. PDF 다운로드
```

### 방법 3: pandoc 사용
```bash
# pandoc 설치 (Mac)
brew install pandoc

# PDF 변환
pandoc 사업계획서_닥터푸드.md -o 사업계획서_닥터푸드.pdf
```
