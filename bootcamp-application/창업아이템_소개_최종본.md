# 닥터푸드(Dr.Food) 창업아이템 소개

## 📋 신청서용 창업아이템 소개 (최종본)

---

## 버전 1: 기본 버전 (3줄)

```
닥터푸드(Dr.Food)는 요양시설의 식사·투약·건강상태·보호자 소통 데이터를 통합 관리하는 고령자 케어 푸드 플랫폼입니다. 기존 요양현장에서 분절적으로 운영되던 돌봄 정보를 관리자(시설 운영자)·간병인(현장 케어 인력)·보호자(가족) 3개 사용자 앱으로 연결하여, 하나의 디지털 워크플로우 안에서 기록·확인·소통이 이루어지도록 설계한 SaaS형 서비스입니다. 사진 한 장으로 식사 기록이 자동 완료되고, AI가 개인 맞춤 식단을 추천하며, 보호자 앱으로 실시간 소통하여 요양원 운영 효율을 75% 향상시키고 식자재 비용을 38% 절감합니다.
```

**글자 수**: 약 300자
**특징**: 포괄적, 전문적, 모든 정보 포함

---

## 버전 2: 고객 중심 버전 (3줄)

```
닥터푸드(Dr.Food)는 요양원 운영자가 가장 고민하는 '기록 업무 과중', '보호자 응대 부담', '투약 누락 위험' 3가지 문제를 해결하는 AI 기반 통합 케어 플랫폼입니다. 관리자·간병인·보호자 3개 역할별 앱으로 식사·투약·건강 데이터를 하나의 워크플로우로 연결하여, 사진 한 장으로 식사 기록이 완료되고 AI가 32명 전원의 개인 맞춤 식단을 자동 생성하며 보호자 앱으로 실시간 공유합니다. 하루 2시간 기록 시간을 75% 단축하고, 식자재 공동구매로 월 100만원을 절감하며, 보호자 전화 문의를 70% 감소시켜 요양원 운영 효율을 혁신합니다.
```

**글자 수**: 약 320자
**특징**: 문제 해결 중심, 구체적 수치 강조

## 버전 3: 간결 버전 (2줄)

```

```

**글자 수**: 약 200자
**특징**: 간결, 핵심만 전달, 수치 강조

## 버전 4: 타겟 고객 강조 버전 (3줄)

```
닥터푸드(Dr.Food)는 중소형 요양원(30-50명 규모) 운영자를 위한 AI 기반 통합 케어 플랫폼으로, 식사·투약·건강·보호자 소통 데이터를 하나의 디지털 워크플로우로 연결합니다. 관리자·간병인·보호자 3개 역할별 앱을 통해 사진 한 장으로 식사 기록이 자동 완료되고, AI가 기저질환별 맞춤 식단을 생성하며, 보호자 앱으로 실시간 공유하여 운영 효율을 극대화합니다. 전국 3,500개 중소형 요양원을 대상으로 하루 2시간 기록 시간 절감, 월 100만원 식자재 비용 절감, 보호자 만족도 80% 향상을 목표로 합니다.
```

**글자 수**: 약 300자
**특징**: 타겟 고객 명확, 시장 규모 언급

## 버전 5: 차별화 강조 버전 (3줄)

```
닥터푸드(Dr.Food)는 국내 유일의 3-Role 통합 케어 플랫폼으로, 요양시설의 관리자·간병인·보호자를 하나의 디지털 워크플로우로 연결하여 식사·투약·건강 데이터를 실시간 공유합니다. 기존 솔루션과 달리 사진 한 장으로 AI가 자동으로 식사를 분석하고, 32명 전원의 기저질환별 맞춤 식단을 자동 생성하며, 식자재 공동구매로 비용을 38% 절감하는 차별화된 가치를 제공합니다. 실제 작동하는 제품을 GitHub Pages에 배포 완료하여 즉시 체험 가능하며, 한국에서 검증 후 미국 시장($436B 규모) 진출을 목표로 합니다.
```

**글자 수**: 약 320자
**특징**: 차별화 포인트 강조, 글로벌 비전
