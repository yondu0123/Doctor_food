# 💻 로컬 개발 환경 설정

> DoctorFood 로컬 개발 환경 구축 가이드

---

## 📋 시스템 요구사항

### 필수 소프트웨어
- **Python**: 3.9 이상
- **Node.js**: 16 이상 (선택사항, 프론트엔드 빌드 도구 사용 시)
- **Git**: 최신 버전
- **텍스트 에디터**: VS Code, PyCharm 등

### 권장 사양
- **OS**: macOS, Linux, Windows 10/11
- **RAM**: 4GB 이상
- **디스크**: 2GB 이상 여유 공간

## 🚀 설치 가이드

### 1. 저장소 클론

```bash
# 프로젝트 클론
git clone https://github.com/your-username/doctorfood-app.git
cd doctorfood-app

# 또는 이미 다운로드한 경우
cd doctorfood-app
```

---

### 2. 백엔드 설정

#### 2.1 가상환경 생성 및 활성화

```bash
cd backend

# 가상환경 생성
python -m venv venv

# 가상환경 활성화
# macOS/Linux:
source venv/bin/activate

# Windows:
venv\Scripts\activate

# 가상환경 활성화 확인 (프롬프트에 (venv) 표시됨)
```

#### 2.2 패키지 설치

```bash
# 필수 패키지 설치
pip install -r requirements.txt

# 설치 확인
pip list
```

**주요 패키지:**
- `fastapi`: 웹 프레임워크
- `uvicorn`: ASGI 서버
- `sqlalchemy`: ORM
- `pydantic`: 데이터 검증
- `python-jose`: JWT 처리
- `passlib`: 비밀번호 해싱
- `google-generativeai`: Gemini AI
- `httpx`: HTTP 클라이언트

#### 2.3 환경 변수 설정

```bash
# .env.example 복사
cp .env.example .env

# .env 파일 편집
nano .env  # 또는 vi, code 등
```

**.env 파일 내용:**
```bash
# 필수 설정
SECRET_KEY=your-secret-key-here-min-32-chars
GEMINI_API_KEY=your-gemini-api-key

# 카카오 OAuth (선택)
KAKAO_CLIENT_ID=your_kakao_client_id
KAKAO_CLIENT_SECRET=your_kakao_client_secret
KAKAO_REDIRECT_URI=http://localhost:8000/api/auth/kakao/callback

# 기타 설정 (기본값 사용 가능)
DATABASE_URL=sqlite:///./doctorfood.db
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=10080
```

**SECRET_KEY 생성:**
```bash
python -c "import secrets; print(secrets.token_urlsafe(32))"
```

#### 2.4 데이터베이스 초기화

```bash
# 데이터베이스 테이블 생성
python -c "from app.core.database import init_db; init_db()"

# 또는 서버 실행 시 자동 생성됨
```

#### 2.5 백엔드 서버 실행

```bash
# 개발 서버 실행 (자동 재시작 활성화)
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 또는 간단하게
python -m uvicorn app.main:app --reload
```

**서버 실행 확인:**
- API 문서: http://localhost:8000/docs
- 대체 문서: http://localhost:8000/redoc
- Health Check: http://localhost:8000/health

## 🐛 문제 해결

### 1. 가상환경 활성화 실패

**Windows PowerShell 오류:**
```powershell
# 실행 정책 변경
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# 가상환경 재활성화
venv\Scripts\activate
```

### 2. 패키지 설치 오류

```bash
# pip 업그레이드
pip install --upgrade pip

# 캐시 삭제 후 재설치
pip cache purge
pip install -r requirements.txt
```

### 3. 포트 충돌

```bash
# 8000번 포트 사용 중인 프로세스 확인
# macOS/Linux:
lsof -i :8000

# Windows:
netstat -ano | findstr :8000

# 다른 포트로 실행
uvicorn app.main:app --reload --port 8001
```

### 4. CORS 오류

```bash
# .env 파일에 프론트엔드 URL 추가
ALLOWED_ORIGINS=http://localhost:5500,http://127.0.0.1:5500
```

### 5. 데이터베이스 오류

```bash
# 데이터베이스 파일 삭제 후 재생성
rm doctorfood.db
python -c "from app.core.database import init_db; init_db()"
```
