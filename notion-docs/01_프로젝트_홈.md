## 📝 개발 환경

### 서버 실행
```bash
# 백엔드 실행
cd backend
source venv311/bin/activate
uvicorn app.main:app --reload --port 8000

# 프론트엔드 실행
cd frontend
python3 -m http.server 3000
```

### 주요 디렉토리
```
doctorfood-app/
├── backend/          # FastAPI 백엔드
│   ├── app/
│   │   ├── api/      # API 엔드포인트
│   │   ├── models/   # 데이터베이스 모델
│   │   ├── services/ # 비즈니스 로직
│   │   └── auth/     # 인증 관련
│   └── venv311/      # Python 가상환경
├── frontend/         # 프론트엔드
│   ├── pages/        # HTML 페이지
│   └── shared/       # 공통 리소스
└── docs/             # 문서
```
