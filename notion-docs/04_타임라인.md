# 📅 타임라인

> 프로젝트 일정을 시간 순서로 관리

---

## 📌 사용 방법

### 노션에서 타임라인 뷰 생성
1. 칸반 보드 데이터베이스에서 "+ Add a view" 클릭
2. "Timeline" 선택
3. 뷰 이름: `타임라인`
4. **Start date**: `시작일`
5. **End date**: `마감일`
6. **Color by**: `우선순위`
7. **Group by**: `카테고리` (선택사항)

---
