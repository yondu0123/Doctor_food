## 📌 사용 방법

### 노션에서 테이블 뷰 생성
1. 칸반 보드 데이터베이스에서 "+ Add a view" 클릭
2. "Table" 선택
3. 뷰 이름: `작업 목록`
4. **Sort**: 
   - 1순위: `우선순위` (Descending)
   - 2순위: `마감일` (Ascending)
5. **Filter**: `상태` is not `완료` (선택사항)

## 💡 작업 목록 활용 팁

### 1. 정렬 활용
- **우선순위 + 마감일**: 가장 중요하고 급한 작업 먼저
- **카테고리 + 상태**: 영역별 진행 상황 파악
- **진행률**: 거의 완료된 작업 먼저 마무리

### 2. 그룹화
- **상태별**: 칸반 보드와 유사한 뷰
- **카테고리별**: 영역별 작업 관리
- **우선순위별**: 중요도 기준 관리

### 3. 검색
- 작업명으로 빠른 검색
- 태그로 관련 작업 찾기
- 설명 내용으로 검색
