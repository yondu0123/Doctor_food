## 📌 사용 방법

### 노션에서 칸반 보드 생성
1. `NOTION_TASKS.csv` 파일을 노션에 Import
2. 데이터베이스 뷰에서 "+ Add a view" 클릭
3. "Board" 선택
4. **Group by**: `상태` 선택
5. **Sort**: `우선순위` (Descending)

## 🎯 작업 상태 정의

### 📋 To Do (예정)
- 아직 시작하지 않은 작업
- 우선순위에 따라 정렬
- 다음에 할 작업 선택

### 🚧 진행중 (In Progress)
- 현재 작업 중인 항목
- 진행률 표시 (0-100%)
- 최대 2-3개 유지 권장

### ✅ 완료 (Done)
- 완료된 작업
- 완료일 자동 기록
- 주간 리뷰 시 확인

### 🔒 보류 (On Hold)
- 일시 중단된 작업
- 블로커 또는 의존성 대기
- 재개 조건 명시

## 💡 칸반 보드 활용 팁

### 1. 작업 이동
- 드래그 & 드롭으로 상태 변경
- 작업 시작 시 "To Do" → "진행중"
- 완료 시 "진행중" → "완료"

### 2. 진행률 업데이트
- 작업 카드 클릭
- 진행률 속성 업데이트 (0-100%)
- 메모에 진행 상황 기록

### 3. 우선순위 관리
- 🔴 High: 긴급하고 중요
- 🟡 Medium: 중요하지만 여유 있음
- 🟢 Low: 나중에 해도 됨

### 4. 필터 활용
```
이번 주 작업만 보기:
Filter: 마감일 is within this week

긴급 작업만 보기:
Filter: 우선순위 is 🔴 High
AND 상태 is not 완료

내 작업만 보기:
Filter: 담당자 is Me
```
